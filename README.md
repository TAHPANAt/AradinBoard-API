# AradinBoard API

## 📋 Project Overview
AradinBoard คือระบบเว็บบอร์ดขนาดเล็กที่สร้างขึ้นเพื่อสาธิตการทำงานของ End-to-End CI/CD Pipeline ตามโจทย์วิชา ENG23 3074

## 🛠️ Tech Stack & Pipeline
- **Language:** Go (Gin framework) 
- **CI/CD:** Jenkins
- **Infrastructure:** Terraform & Ansible
- **Orchestration:** Kubernetes
- **Monitoring:** Prometheus & Grafana
